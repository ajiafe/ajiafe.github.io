快速安装 hugo #

访问hugo 官网-快速开始

  1. 创建网站
  2. 添加内容
  3. 配置网站
  4. 发布网站

首先安装 hugo 不同的操作系统安装方式不同，以下以 Mac 为主

brew install hugo

1. 创建网站 #

hugo new site quickstart
cd quickstart
git init
git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/congo
echo "theme = 'congo'" >> hugo.toml
hugo server

congo 是 hugo 的一个主题，想要使用不同的主题，修改相应的名字即可。可参考官方主题地址

注意问题：主题配置必须写在hugo.toml文件中，若写在config.toml配置中不起作用、会报错

执行完成上面的内容你的网站就可以在本地正常运行起来。

2. 添加内容 #

使用 hugo new posts/article.md 命令添加文章

hugo new posts/my-first-post.md

在 content/posts 目录下会生成 my-first-post.md 文件并生成一下内容

---
title: "My First Post"
date: 2022-11-20T09:03:20-08:00
draft: true
---

draft: true 表示草稿,不会再网站上显示。想要在开发环境下展示，使用以下命令：

hugo server -D

可以通过 http://localhost:1313/posts/my-first-post/ 路径在本地访问该文章。

3.配置网站 #

在根目录下的 hugo.toml 文件下添加如下配置

theme = 'congo'
baseURL = '你的网站域名'
languageCode = 'en-us'
title = '标题'

此时网站首页则会同步以上配置。

4. 部署网站 #

在项目中执行 hugo 命令，会自动生成 Public 目录，该目录下的文件就是站点的打包后的静态文件。可以将该文件存放在自己的服务器上，并通过Nginx配置来访问。

该步骤有很多个方案可以参考，具体可看官网教程，不再赘述。
